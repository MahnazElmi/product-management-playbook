# Product Launch Checklist

A practical checklist for launching digital products.

---

# Phase 1 — Strategy

- Define the target audience
- Validate the problem
- Define the value proposition
- Identify competitors
- Set launch goals
- Define success metrics

Deliverables

- Positioning
- Messaging
- Launch Goals
- KPI Dashboard

---

# Phase 2 — Product Readiness

- Feature complete
- QA completed
- Analytics implemented
- Error monitoring enabled
- Documentation published

Checklist

- GA4 / PostHog events
- Crash monitoring
- Feature flags
- Rollback plan

---

# Phase 3 — Marketing

- Landing page
- Product Hunt
- Blog post
- Email campaign
- Social media
- Press kit

---

# Phase 4 — Sales

- Pricing finalized
- Sales deck
- Demo environment
- FAQ
- Objection handling

---

# Phase 5 — Customer Success

- Help Center
- Tutorials
- Support playbooks
- Onboarding emails

---

# Phase 6 — Launch Day

Monitor

- Traffic
- Signups
- Activation
- Revenue
- Errors
- Customer feedback

---

# Phase 7 — Post Launch

Review

- What worked?
- What failed?
- User feedback
- Feature requests
- Roadmap updates
